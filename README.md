# CursorVision

CursorVision is a webcam-based desktop interaction system for Linux.

## Planned Features

- Head-controlled cursor
- Eye tracking
- Blink gestures
- Eyebrow gestures
- Gaze-triggered UI
- Linux desktop integration

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
